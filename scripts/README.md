# Scripts Directory

This directory contains R scripts for data analysis and modeling.

## Recommended naming convention

- `01_data_import.R` - Data loading and initial exploration
- `02_data_cleaning.R` - Data cleaning and preprocessing
- `03_exploratory_analysis.R` - Exploratory data analysis
- `04_modeling.R` - Statistical modeling
- `05_visualization.R` - Creating plots and figures

## Guidelines

- Number scripts in the order they should be run
- Use descriptive names
- Include comments and documentation within scripts
